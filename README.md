# work-dev

Professional development work — mobility data, open data tools & spatial analysis.

Code and tools from my professional research activities at Bauhaus-Universität Weimar.

## What lives here

**Mobility & transport data**  
Scripts and tools from my work at the Chair of Transport System Planning — processing
open mobility data (GTFS, OSM, sensor feeds), building interactive exploration platforms,
and supporting EDIH.TH consulting projects.

**Urban spatial analysis**  
GIS workflows, network analysis pipelines, and data tools developed for spatial-economic
research and urban data science.

**DeCodingSpaces contributions**  
Utilities and C# extensions from my work on the DeCodingSpaces Toolbox — a Grasshopper
plugin for computational urban analysis.

## Stack
